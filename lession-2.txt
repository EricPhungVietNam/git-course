Câu 1:
   git init               : đưa folder vào Git quản lý
   git add <file_name>    : chọn file cụ thể để đưa vào vùng Staging
   git add .              : chọn tất cả file để đưa vào vùng Staging
   git commit -m "message": snapshoot file đã chọn lưu vào Local repository.
